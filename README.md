# Cogniscan 🚀

**Your Personal Tech + Business Intelligence Radar**

Cogniscan helps you stay ahead of the curve by delivering **personalized tech, business, and career insights** tailored to your background.

Instead of consuming generic news, Cogniscan filters and curates information **relevant to you** — helping you spot opportunities faster.

---

## ✨ Features (V1.0)

* 📄 Resume-based personalization
* 📰 Curated tech & business news
* 🤖 AI-powered summarization (via Mistral)
* 📬 Newsletter-style output
* ⚡ Simple CLI-based workflow

---


## 🛠️ Tech Stack

* Python
* Mistral API (LLM)
* NewsAPI
* CLI-based pipeline

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone <your-repo-link>
cd cogniscan
```

### 2. Create `.env` file

Add your API keys:

```
MISTRAL_KEY=your_mistral_key
NEWSORG_API_KEY=your_newsapi_key
```

### 3. Run the project

```bash
cd backend/packages/src
python agent_v1.py --resume <path_to_resume>
```
